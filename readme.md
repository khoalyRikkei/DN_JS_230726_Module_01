# Layout structure

## Khái niệm về Layout

- Tạo bố cục cho website
- Dùng các thẻ ngữ nghĩa để định nghĩa layout: header, footer, main, aside, nav,...

## Position

- Mặc định là: position: static
- Các position khác lưu ý đến top: so với viền trên, buttom: so với border dưới, left: so với border trái, right: so với border phải. top, left ưu tiên hơn bottom, right khi cả 2 đều được set
- position: absolute -> bắt buộc phải có thẻ cha bộc ở ngoài, và postion thẻ cha phải có và khác static. Nếu không set, mặc định thẻ cha là body
